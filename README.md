# Simple Terminal

A simple terminal-based Python application built as a first project while learning Python and Linux.
My first project, a terminal-based Python application.

Includes:
- A command-based terminal interface
- Number guessing game
- Calculator
- Wordle-style word guessing game
- Coloured terminal output
## Requirements
- **Python 3.12 or newer**
- The Python library **rich**

Check your python version with:
python3 --version

## Setup and Run
Without virtual environment:
pip install rich
python terminal.py

With virtual environment:
python3 -m venv .venv
source .venv/bin/activate
pip install rich
python terminal.py
